### 案例2: Map使用的进步

**题目1-20**: 基本使用
```javascript
// 使用Map但不够优雅
const map = new Map()
for (let word of words) {
    if (map.has(word)) {
        map.set(word, map.get(word) + 1)
    } else {
        map.set(word, 1)
    }
}
```

**题目21-30**: 优雅使用
```javascript
// 简洁的Map使用

arr.forEach(i => !map.has(i) ? map.set(i, 1) : map.set(i, map.get(i) + 1))
```

**改进**: 代码更简洁，逻辑更清晰 ✅
